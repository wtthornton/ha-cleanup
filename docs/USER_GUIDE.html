🎯 Overview

ha-cleanup is a powerful, mobile-first web application designed to analyze and visualize Home Assistant historical data. Built with modern web technologies, it provides an intuitive interface for exploring sensor data, events, and system metrics.

✨ Key Benefits:
  • Mobile-optimized responsive design
  • Real-time data visualization
  • Advanced filtering and search capabilities
  • Performance-optimized with code splitting
  • Secure InfluxDB integration

🚀 Getting Started

Prerequisites

🔌 InfluxDB Access

Ensure you have access to an InfluxDB instance with Home Assistant data

🌐 Modern Browser

Chrome, Firefox, Safari, or Edge with JavaScript enabled

📱 Device Ready

Works on desktop, tablet, and mobile devices

Quick Start Steps

  1. Open the application in your web browser
  2. Configure your InfluxDB connection settings
  3. Select your desired time range for data analysis
  4. Start exploring your Home Assistant data!

🔧 Troubleshooting

Common Issues & Solutions

❌ Connection Issues

If you can't connect to InfluxDB:

  • Verify your InfluxDB URL and port
  • Check that your token has read permissions
  • Ensure the bucket name is correct
  • Check network connectivity and firewall settings
⚠️ Performance Issues

If the app feels slow:

  • Check your internet connection speed
  • Try reducing the time range for data queries
  • Clear browser cache and cookies
  • Update to the latest browser version
